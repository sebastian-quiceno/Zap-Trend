# Zap-Trend
Proyecto personal de repaso, el cual va a manejar diferentes tecnologias a nivel de base de datos, backend y frontend.
## PostgreSQL
Este va a ser el motor de base de datos, el cual va a ser utilizado por medio de JPA e hibernante, va a ser usado por medio de docker
## SpringBoot y Spring Security
Esta va a ser la tecnologia usada a nivel de backend el cual va a permitir simplificar la interaccion con la base de daatos, crear API rest y mantener una buena seguridad (por medio de Spring Security)
## React con Vite
Este va a ser usado para el front end
